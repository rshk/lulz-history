What's this?

You can login using GitHub and explore projects history with lolcommits images.

How do I lulz-enable my repo?

  1. Install lolcommits and enable it on your repo
  2. Commit your lolcommits pictures (you find them in ~/.lolcommits) in a branch named "lulz-pics"
  3. There isn't a 3, since you're done. Congrats!
