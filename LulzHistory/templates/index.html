lolcommits + GitHub == more lulz!

Navigate project history through contributor's faces!

What's this?

Ever wanted to see the faces of the people who wrote the software you're using and trusting every day? Now you can easily, thanks to lolcommits, GitHub and this app!

How do I submit my pictures?

Just create a repository named my-lulz-pics and commit your ~/.lolcommits directory into it.

Lulz-history will look for pictures in all the subfolders of the my-lulz-pics repository owned by the commit author.

You can also put pictures in a branch named lulz-pics, in the same repository as the code, but I opted out from that solution as it would require more work to set up and forces to keep around / merge all the pictures from all the projects contributors.
